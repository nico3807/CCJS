Comment ça marche ?

1

2

3

Clique sur ▶ Tester le code pour voir le résultat s'exécuter en direct.

4

Clique sur ✔ Soumettre ma réponse : l'IA évalue ton code et te donne un retour immédiat.

5

Si tu es bloqué, utilise le bouton Assistant pour obtenir des indices personnalisés.
